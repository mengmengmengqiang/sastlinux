稍稍深入一点认识Shell

Shell地位相当于windows上的cmd，或者说是PowerShell，

首先Shell本身也是一个程序，或者说只是一个程序，

Shell的英文直译是“壳”，是相对于“核”的， “核”是看不见的然而“壳”是看的见的， 也就是Shell其实就是操纵Linux的人直接控制的东西，

说白了就是那个输入命令的界面， 我们的一个一个命令实际上都是传给Shell， 然后由Shell产生新进程去调用其他程序，

就是一个命令的解释器，类似于python的解释器，一行一行的输入和解释，如果写入一个文件再去解释就是脚本


解释器也有不同的，像python的解释器也有好几个不同的实现，linux也是

用的最广泛的是bash,其他的还有zsh,csh之类

不同的解释器，大多是相同的，但是还是有点不同的，有些语法方面会有差别,像查看上一个任务的返回码

.bashrc文件-进入各种SHELL时都会先执行一些脚本

变量

类似与python一样，我们可以定义自己的变量，用echo查看变量以及做一些运算

用set查看定义的变量

unset删除变量

export把设定传到子shell

一些特殊的变量

  • HOME-用户的家目录，比如直接的cd就默认到家目录
  • USER-用户
  • SHELL
  • LANG
  • PS1-命令提示符
  • PATH
